# sitecore-docker

This repository contains utilities and `Dockerfile`s to create images for the Sitecore platform.

The following images will be built upon execution of `Build-ScDockerImages`:

#### Base Images

* scbase:aspnet
* scbase:netcore
* scbase:netframework

#### Application Images

* sitecore:latest
* xconnect:latest
* xconnect.indexworker:latest
* xconnect.maengine:latest
* xconnect.processingengine:latest
* identity
